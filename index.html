💳 Cobro con terminal (tarjeta)

NovaPOS no procesa pagos con tarjeta por sí mismo: esto requiere una cuenta de comercio con un procesador (Conekta, Stripe o Square) y, en la mayoría de los casos, su propia terminal física certificada.

📋 Cómo se conecta (guía general)
1. Da de alta un contrato de comercio con Conekta, Stripe o Square (piden RFC y datos fiscales del negocio).
2. Cada procesador certifica su propia terminal física o modalidad "tap to pay" — no es un botón que se agregue a cualquier página.
3. Las llaves secretas del procesador nunca deben ir en este archivo HTML/JS — cualquiera con el código fuente podría verlas y usarlas. Deben manejarse desde un servidor propio.
4. Esto implica agregar un backend al proyecto (NovaPOS hoy usa Google Sheets, que no es apto para procesar cobros con tarjeta).
5. Es un proyecto de desarrollo aparte, no una casilla que se activa aquí — si te interesa, contáctanos para cotizarlo a detalle.

📋 Instrucciones de configuración

2. Ve a Extensiones → Apps Script → pega el código del backend.
3. Ejecuta setupSheets para crear las pestañas.
4. Implementa como Aplicación Web → Cualquier persona → copia la URL.
5. Pégala arriba y guarda.
